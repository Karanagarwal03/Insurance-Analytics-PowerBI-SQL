# 🏦 Insurance Analytics Dashboard (SQL + Power BI)

## 🚀 Project Overview

This project is an **end-to-end Insurance Analytics solution** built using **SQL Server and Power BI**, designed to analyze policy performance, claims behavior, and customer feedback.

It combines **data engineering, data modeling, and visualization** to deliver actionable business insights.

---

## 🎯 Objectives

* Analyze insurance policy performance
* Track claims and settlement trends
* Understand customer demographics
* Perform sentiment analysis on customer feedback
* Implement secure data access using RLS

---

## 🛠️ Tech Stack

* **Microsoft SQL Server** (Data Storage & Querying)
* **Power BI Desktop & Service**
* **Power Query (ETL)**
* **DAX (Data Analysis Expressions)**
* **Excel / CSV Dataset**

---

## 🔄 Project Workflow

1. Data imported into **SQL Server**
2. Data cleaned & transformed using **Power Query**
3. Data modeled in Power BI
4. DAX measures created for KPIs
5. Interactive dashboard built
6. Report published to Power BI Service
7. Scheduled refresh configured
8. Row-Level Security (RLS) implemented
9. Sentiment analysis performed on feedback data

## 💼 Business Impact

This project simulates how an insurance company can leverage data analytics to improve decision-making and operational performance.

* 📉 **Reduce Claim Rejections**
  High number of rejected claims indicates process gaps. By analyzing rejection patterns, companies can refine claim validation rules and reduce unnecessary rejections.

* ⚙️ **Improve Claim Processing Efficiency**
  Insights into claim status distribution (Pending, Settled, Rejected) help identify bottlenecks and optimize processing workflows.

* 👥 **Enhance Customer Experience**
  Sentiment analysis highlights customer pain points (e.g., delays, confusion), enabling targeted improvements in service quality.

* 📊 **Optimize Policy Offerings**
  Understanding which policy types generate the highest premiums helps businesses focus on high-performing products.

* 🎯 **Target High-Value Customer Segments**
  Age-group and demographic analysis allows companies to design better marketing and retention strategies.

* 🔐 **Secure Data Access with RLS**
  Ensures that sensitive customer and policy data is only accessible to authorized users, improving compliance and data security.

* 📈 **Support Data-Driven Decision Making**
  Enables stakeholders to make informed decisions based on real-time insights rather than assumptions.
